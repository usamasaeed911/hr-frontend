# HR System Frontend

React + Vite HR Management System.

## Features
- Dashboard
- Employees Management
- Leave Requests
- Attendance
- Payroll
- Expenses
- Recruitment
- Documents
- Super Admin Panel
- PWA Support

## Tech Stack
- React 18
- Vite
- Supabase
- Lucide Icons
